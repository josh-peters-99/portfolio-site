Linked List Reversal

Pattern: Linked Lists

Problem: Reverse a singly linked list. (This problem has two solutions - ) 

( 1 ) --> ( 2 ) --> ( 4 ) --> ( 7 ) --> ( 3 )

                      ↓

( 3 ) --> ( 7 ) --> ( 4 ) --> ( 2 ) --> ( 1 )

Key Insight:

Time Complexity: O(n) where n is the number of nodes in the linked list.

Space Complexity: O(1)
