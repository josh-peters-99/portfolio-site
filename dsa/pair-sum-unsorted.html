Pair Sum - Unsorted

Problem

Given an array of integers, return the indexes of any two numbers that add up to a target. The order of the indexes in the result doesn't matter. If no pair is found, return an empty array.

Example

        Input: nums = [-1, 5, 7, 4], target = 9
        Output: [1, 3]

Steps to Solution

  1. Initialize a hash map:
    This will store numbers from the array as keys and their indices as values.
  2. Iterate through the input array using a loop:
    Go through each number in the array one by one, keeping track of the current index.
  3. For each number, calculate the difference:
    Subtract the current number from the target value.
  4. Check if the difference is already a key in the hash map:
    If it is, return an array with two elements: the index of the difference in map and the current index.
    It it is not, continue to the next step.
  5. Add the current number and its index to the hash map:
    Store it as a key-value pair.
  6. If the loop completes without finding a matching pair:
    Return an empty array.
